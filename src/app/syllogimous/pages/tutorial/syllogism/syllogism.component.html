How to Play Syllogism

Each syllogism question starts with a set of premises. For example:

  • Some Planet is Soup
  • No Freezer is Turtle
  • No Freezer is Planet
  • All Turtle is Planet

The game will present a conclusion that you must evaluate based on the premises. For example:

Conclusion: "Some Freezer is Soup."
