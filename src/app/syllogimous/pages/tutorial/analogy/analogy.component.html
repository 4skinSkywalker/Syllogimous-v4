How to Play Analogy

Each analogy question begins with a set of statements that define the relationships between different items. For example:

  • Brush is the same as Desk
  • Desk is opposite of Bread
  • Skirt is opposite of Heart
  • Bread is the same as Skirt

After reviewing the relationships, you'll be presented with an analogy that you need to evaluate. For example:

Conclusion: "Heart to Bread is the same as Skirt to Desk."

Based on your analysis, click the True button if you believe the given relationship is correct or the False button if you think it is not.
